FormDropdown component

Files in this folder:

- `FormDropdown.tsx` — the component that renders a select-style dropdown integrated with react-hook-form.
- `FormDropdown.test.tsx` — unit tests for the component that mock the UI primitives and verify placeholder, selected label, and onValueChange behavior.
- `FormDropdown.mocks.ts` — sample options and groups used by tests.

Run tests for this component:

```bash
npm run test -- src/components/forms/FormDropdown/FormDropdown.test.tsx -i
```
